Books

Manage books.
  • GET /api/v1/books
    Get all books
  • GET /api/v1/books/id
    Get a single book with given id
  • POST /api/v1/books
    Create a new book
  • PUT /api/v1/books/id
    Update the book with given id
  • DELETE /api/v1/books/id
    Delete the book with given id
